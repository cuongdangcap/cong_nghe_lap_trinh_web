# Bài tập buổi 6 - CRUD Controller kết hợp Layout và Razor View Engine

## Chủ đề

Quản lý sinh viên.

## Chức năng

* Hiển thị danh sách sinh viên.
* Xem chi tiết sinh viên.
* Thêm sinh viên mới.
* Sửa thông tin sinh viên.
* Xóa sinh viên.
* Validation dữ liệu bằng Data Annotation.

## Công nghệ sử dụng

* ASP.NET Core MVC
* Razor View Engine
* Layout (_Layout.cshtml)
* Bootstrap

## Dữ liệu

Dữ liệu được lưu bằng List giả lập trong chương trình, chưa sử dụng cơ sở dữ liệu.
## Hướng dẫn chạy project

Mở Terminal tại thư mục project:

dotnet restore

dotnet run

Sau đó truy cập:

[http://localhost:5103/Student](http://localhost:5103/Student)

(5103 là cổng được ASP.NET Core tự sinh)
